WildFly Configuration API

The WildFly configuration API is a generated object model representing the set of configuration options for the WildFly application server. Clients such as WildFly Swarm may use this to configure a running instance of WildFly via a fluent Java API.

For more information, see the README or read the WildFly Swarm User's Guide.
